hexo deploy 免密登陆

发表于 2019-06-30 | 分类于 hexo

1.将https模式转换为ssh模式

2.设置SSH keys

（1）看看 C:\Users\Administrator.ssh里面是否有文件
（2）有就删了
（3）生成密匙 输入指令

1.git config –global user.email “862396534@qq.com” # 填写你github注册并且验证的邮箱

2.git config –global user.name “mykonons” # github 用户名

3.ssh-keygen # 一直按Enter键就行

（4）打开 C:\Users\Administrator.ssh\id_rsa.pub 文件 复制所有
（5）github 用户名头像旁的向下三角号-》setting
（6）personal settings -》 ssh and gpg keys-》add keys
（7） 将第四步复制的全粘贴到key里
（8） ssh -T git@github.com
（9）编辑_config.yml文件，修改文件里面的deploy，
deploy:
type: git
repo: git@github.com:mykonons/mykonons.github.io.git
branch: master

hexo的简单使用

发表于 2019-06-30 | 分类于 hexo

1.进入hexo工作目录

  1. cd d:
  2. cd blog/myblog

2.hexo n “文章名”

3.编辑文章

4.hexo s 启动服务器，登陆localhost

5.hexo clean 清理

6.hexo g 生成文章

7.hexo d 推到远程服务器
